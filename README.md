## Installation

Install Drink Helper with npm.

1.Clone repository:

```bash
git clone https://github.com/Imperface/drinkHelper.git
```

2.Open folder:

```bash
cd drinkHelper
```

3.Install node modules:

```bash
npm i
```

4.Run the app:

```bash
npm run dev
```
